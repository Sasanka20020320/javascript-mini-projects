# Resident Registry System

A JavaScript mini-project simulating a Grama Niladhari resident management system.

## Features

- Add Resident
- Find Resident by NIC
- Update Resident
- Delete Resident
- Generate Reports
- Input Validation

## Concepts Practiced

- Arrays
- Objects
- CRUD Operations
- Functions
- Validation
- Error Handling

## Future Improvements

- Store data in JSON files
- Convert to Node.js
- Add Express API
- Connect MongoDB
