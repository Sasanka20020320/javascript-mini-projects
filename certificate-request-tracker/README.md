# Certificate Request Tracker

A JavaScript mini project that simulates a certificate request management system for a Grama Niladhari office.

## Features

* Create Certificate Requests
* Validate Request Data
* View All Requests
* Find Request by ID
* Approve Requests
* Reject Requests
* Filter Requests by Status
* Generate Request Dashboard Statistics

## Concepts Used

* Arrays and Objects
* Functions
* CRUD Operations
* Data Validation
* Array Methods (`find`, `filter`, `reduce`)
* Business Workflow Logic
* Dashboard Reporting

## Example Workflow

1. A resident submits a certificate request.
2. The request is stored with a status of `Pending`.
3. A GN officer can:

   * Approve the request
   * Reject the request
4. Dashboard statistics are generated to show the number of:

   * Pending Requests
   * Approved Requests
   * Rejected Requests
